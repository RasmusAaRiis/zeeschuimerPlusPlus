# 🏴‍☠️ Zeeschuimer++

A branch of [Zeeschuimer](https://github.com/digitalmethodsinitiative/Zeeschuimer) with additional features:
 - Export as JSON
 - Instagram comments
 - Instagram followers count
 - Instagram following count
 - Instagram profile bio

### Installation

Using Firefox Developer Edition:
1) Navigate to 'about:config'
2) Set 'xpinstall.signatures.required' to 'false'
3) Download [Zeeschuimer++.xpi](https://github.com/RasmusAaRiis/zeeschuimerPlusPlus/releases/download/v1.13.6/Zeeschuimer++.xpi)
